# Dengue NS2B-NS3 Protease Inhibitor Screening — QSAR + Conformal Prediction

Curated QSAR benchmark with conformal prediction for dengue NS2B-NS3 protease inhibitor prioritisation.  
Companion Code for: *Conformal Prediction for Dengue NS2B-NS3 Protease Inhibitor Screening: A Curated QSAR Benchmark* (Ranjan & Sharma, 2026).

## Setup

```bash
pip install -r requirements.txt
```

Requires Python 3.12+, RDKit 2024.03.1.
